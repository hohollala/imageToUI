ITU MCP Server

이미지를 UI/UX 코드로 변환하는 스마트 도구

AI 기반 분석

이미지를 분석하여 UI 요소, 색상 팔레트, 레이아웃을 자동으로 추출합니다.

다중 프레임워크

React, Vue, HTML/CSS, Flutter, React Native 코드를 생성할 수 있습니다.

반응형 디자인

모바일과 데스크톱에 최적화된 반응형 UI 코드를 자동 생성합니다.

디자인 시스템

일관된 색상, 타이포그래피, 컴포넌트 스타일을 적용합니다.

1

ITU-init

프로젝트를 초기화하고 설정 파일을 생성합니다

2

ITU-start

이미지를 업로드하고 UI 코드로 변환합니다

3

ITU-modify

설정을 수정하고 코드를 재생성합니다

4

ITU-status

프로젝트 상태와 히스토리를 확인합니다

ITU 명령어 가이드

ITU-init 프로젝트 초기화 및 설정 파일 생성
# 기본 설정으로 초기화 ITU-init # 커스텀 설정으로 초기화 ITU-init --projectPath ./my-project --framework vue --responsive true

결과: itu_requirements.md 파일 생성 및 프로젝트 디렉토리 설정

ITU-start 이미지 분석 및 UI 코드 생성
# 이미지를 UI 코드로 변환 ITU-start --imagePath ./design-mockup.png # 커스텀 출력 경로 지정 ITU-start --imagePath ./design.png --outputPath ./custom-output

지원 형식: PNG, JPG, BMP

ITU-modify 설정 수정 및 코드 재생성
ITU-modify --imagePath ./design.png --modifications '{ "framework": "react", "theme": "dark", "colorPalette": ["#1a1a1a", "#ffffff", "#007acc"], "typography": { "primaryFont": "Inter, sans-serif", "fontSize": 16 }, "components": { "buttonStyle": "pill", "borderRadius": 12, "shadows": true } }'
ITU-status 프로젝트 상태 확인
# 현재 프로젝트 상태 확인 ITU-status # 특정 설정 파일 확인 ITU-status --requirementsPath ./custom/itu_requirements.md
팁: 복잡한 디자인은 섹션별로 나누어 변환하면 더 정확한 결과를 얻을 수 있습니다.

설치 및 설정

⚠️ 사전 요구사항
Node.js 18.0.0 이상이 설치되어 있어야 합니다.

단계별 설치 가이드

1 프로젝트 다운로드
git clone https://github.com/your-repo/ImageToUI.git
2 프로젝트 폴더로 이동
cd ImageToUI
3 의존성 설치
npm install
4 프로젝트 빌드
npm run build
5 MCP 서버 등록

👤 사용자 레벨 등록 (자동 설치)

사용자 전체에서 사용 가능하도록 등록합니다.

Windows (PowerShell)
claude mcp add image-to-ui -s user -- node "$PWD/dist/index.js"
Windows (CMD)
claude mcp add image-to-ui -s user -- node "[설치경로]/dist/index.js"
macOS/Linux
claude mcp add image-to-ui -s user -- node $(pwd)/dist/index.js

📝 수동 설정: MCP 설정 파일 편집

각 MCP 클라이언트의 설정 파일을 직접 편집하여 Image-to-UI MCP 서버를 추가할 수 있습니다.

Claude Desktop

설정 파일 위치:

  • Windows: %USERPROFILE%\.claude\settings.json
  • macOS/Linux: ~/.claude/settings.json
{ "mcpServers": { "image-to-ui": { "command": "node", "args": ["[설치위치]/dist/index.js", "--stdin"], "env": { "OPENAI_API_KEY": "your-api-key-here", "DEFAULT_OUTPUT_PATH": "./generated", "DEFAULT_FRAMEWORK": "react" } } } }
Cursor

설정 파일 위치:

  • Windows: %APPDATA%\Cursor\User\mcp.json
  • macOS/Linux: ~/.config/Cursor/User/mcp.json
{ "mcpServers": { "image-to-ui": { "command": "node", "args": ["[설치위치]/dist/index.js", "--stdin"], "env": { "OPENAI_API_KEY": "your-api-key-here", "DEFAULT_OUTPUT_PATH": "./generated", "DEFAULT_FRAMEWORK": "react" } } } }
🎉 설치 완료! 이제 MCP 클라이언트에서 Image-to-UI MCP Server를 사용할 수 있습니다.

설치 확인

# 버전 확인 node dist/index.js --version # 기본 기능 테스트 echo '{"jsonrpc":"2.0","method":"tools/list","id":1}' | node dist/index.js --stdin # MCP 서버 등록 확인 claude mcp list

환경 변수 설정 (선택사항)

AI 기반 이미지 분석을 위한 환경 변수를 설정할 수 있습니다:

# .env 파일 생성 # OpenAI API 설정 (고급 이미지 분석용) OPENAI_API_KEY=your-openai-api-key-here # 기본 설정 DEFAULT_OUTPUT_PATH=./generated DEFAULT_FRAMEWORK=react DEFAULT_THEME=light # 이미지 처리 설정 MAX_IMAGE_SIZE=10485760 SUPPORTED_FORMATS=png,jpg,jpeg,bmp # 출력 설정 INCLUDE_STYLES=true RESPONSIVE_DESIGN=true COMPONENT_PREFIX=UI

웹 인터페이스 실행 (선택사항)

브라우저에서 직관적인 UI로 사용할 수 있습니다:

# 웹 서버 실행 npm run web # 브라우저에서 접속 # http://localhost:3000
